Programa que nos permite comprobar la existencia de RootKits en nuestro sistema Debian, distinto a RKHUNTER


Puede emplear diferentes herramientas y técnicas para buscar rootkits en un sistema Debian sin utilizar RKHUNTER. El programa chkrootkit, una herramienta de código abierto que comprueba su sistema en busca de rootkits y otros tipos de malware. Así es como se instala y utiliza:

  1. Abre una terminal en tu sistema Debian
  2. Luego, instala chkrootkit usando el siguiente comando:
              sudo apt install chkrootkit
    3. Una vez que chkrootkit esté instalado, ejecuta un escaneo del sistema con el siguiente comando:
  sudo chkrootkit


La herramienta chkrootkit escaneará su sistema en busca de evidencia de rootkits y otro malware. Es importante recordar que, si bien chkrootkit puede servir como una capa adicional de seguridad, no puede garantizar la detección de todos los rootkits.
