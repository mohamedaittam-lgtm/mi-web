IMPORTANTE

La configuración es la siguiente:

  • DHCP: El Gateway debe ser la IP del server, en mi caso 192.168.2.1
  • WAN: Añadir un adaptador en Bridge al server para que este tenga acceso a la red de redes(Internet)
  • Reenviadores: Se debe añadir dos reenviadores, que son los siguientes:
        • 8.8.8.8
        • 8.8.4.4
