Kali Linux es una distribución de Linux especializada en seguridad informática y pruebas de penetración. 

Para que se utiliza 

Está diseñada para ayudar a los profesionales de seguridad, administradores de sistemas y entusiastas de la seguridad a llevar a cabo pruebas de seguridad en sistemas y redes, identificar vulnerabilidades y evaluar la seguridad de sus propios sistemas. 

10 Aplicaciones que integra


Nmap: Una poderosa herramienta de escaneo de redes que permite descubrir dispositivos, puertos abiertos y servicios en una red.

Wireshark: Un analizador de protocolos de red que permite la captura y el análisis detallado del tráfico de red.

Metasploit Framework: Una plataforma para el desarrollo y ejecución de exploits, lo que permite encontrar y aprovechar vulnerabilidades en sistemas.

Aircrack-ng: Una suite de herramientas para auditoría y pruebas de seguridad en redes inalámbricas, incluyendo la capacidad de romper claves WEP y WPA/WPA2.

Hydra: Una herramienta de fuerza bruta que puede utilizarse para probar contraseñas y credenciales de acceso a sistemas y servicios.

John the Ripper: Otra herramienta de prueba de contraseñas que puede realizar ataques de fuerza bruta y de diccionario para descifrar contraseñas.

Burp Suite: Una herramienta integral de seguridad web utilizada para detectar y explotar vulnerabilidades en aplicaciones web.

Gobuster: Una herramienta de fuerza bruta que se utiliza para descubrir directorios y archivos ocultos en servidores web.

SQLMap: Una herramienta automatizada de inyección SQL que permite identificar y explotar vulnerabilidades de inyección SQL en aplicaciones web.

OpenVAS: El Escáner de Vulnerabilidades Abiertas (OpenVAS) es un marco de pruebas de seguridad que se utiliza para escanear sistemas en busca de vulnerabilidades conocidas.
