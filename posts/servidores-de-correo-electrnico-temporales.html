Los servicios de correo electrónico temporal proporcionan direcciones de correo que se pueden usar temporalmente para recibir mensajes, sin revelar tu dirección de correo principal. Algunos ejemplos son:

1- 10 Minute Mail: Ofrece una dirección de correo electrónico temporal que dura exactamente 10 minutos. Es ideal para confirmaciones de registro y verificación de cuentas.

2- Temp Mail: Proporciona direcciones de correo electrónico temporales que se eliminan automáticamente después de un período de tiempo específico que elijas.

3- Guerrilla Mail: Permite crear una dirección de correo electrónico temporal y usarla para recibir mensajes. También puedes elegir un nombre de usuario personalizado.

4- Mailinator: Ofrece direcciones de correo electrónico temporales en dominios de "mailinator.com". Los mensajes se eliminan después de un período de tiempo.

5- Getnada: Proporciona direcciones de correo electrónico temporales con una amplia variedad de dominios para elegir. Los mensajes caducan después de un tiempo determinado.

6- YopMail: Ofrece direcciones de correo electrónico temporales que están disponibles para su uso público. Puedes elegir una dirección o utilizar una dirección aleatoria.

7- MailDrop: Proporciona direcciones de correo electrónico temporales que se pueden usar para recibir mensajes. Los mensajes se mantienen en la bandeja de entrada durante 24 horas.

8- Jetable: Permite crear direcciones de correo electrónico temporales con la opción de configurar un número específico de redirecciones.

Estos son útiles para registrarse en sitios web sin dar tu correo personal y con eso evitas el spam, pero no son ideales para comunicación a largo plazo o profesional.
