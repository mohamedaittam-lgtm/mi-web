Programa para montar un Servidor SSH en Windows

Para montar un servidor SSH en Windows, puedes utilizar el programa gratuito "FreeSSHd". Se instala y se configura de la siguiente manera:


  1. Descargar el programa desde Aquí.
  2. Ejecutar el archivo y seguir las indicaciones del asistente.
  3. Aparecerá una ventana de configuración. En la pestaña "Server settings," puedes configurar la información básica del servidor SSH, como el puerto en el que deseas que escuche, las cuentas de usuario permitidas, etc.
  4. En la pestaña "Users," puedes agregar cuentas de usuario que tendrán acceso al servidor SSH. Debes definir un nombre de usuario, una contraseña y el directorio raíz al que se permitirá el acceso.
  5. En la pestaña "Host keys," puedes generar o importar claves de host para la autenticación SSH. Esto es opcional, pero se recomienda para mayor seguridad.
  6. En la pestaña "Authentication," puedes configurar las opciones de autenticación, como permitir o restringir ciertos métodos de autenticación.
  7. En la pestaña "SFTP," puedes configurar opciones específicas para el protocolo SFTP, como el acceso a carpetas compartidas.
  8. Una vez que hayas configurado todas las opciones, haz clic en el botón "Start" en la ventana principal de FreeSSHd para iniciar el servidor SSH.
  9. Utiliza un cliente SSH (PuTTY) para conectarte al servidor SSH.
  10. Inicia sesión con las credenciales del usuario configurado anteriormente.
